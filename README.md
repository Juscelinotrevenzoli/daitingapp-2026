Projeto: DatingApp (API)

Descrição: API RESTful desenvolvida como projeto de estudo para prática de desenvolvimento backend com .NET. Fornece uma base para gerenciamento de usuários e operações CRUD, com foco em arquitetura limpa e uso prático de migrações de banco de dados.
Tecnologias:

Plataforma: .NET 8 / ASP.NET Core
Linguagem: C#
ORM: Entity Framework Core (migrations habilitadas)
Banco de Dados: SQLite (arquivo de esquema incluso)
Arquivos de projeto: solution .sln e projeto API (Web API)
Controle de versão: Git
Status:

Finalidade: Projeto de estudo e aprendizagem — não é produção-ready.
Cobertura: Implementa API endpoints básicos e configuração de persistência (veja as migrations em Data/Migrations).
Como executar (exemplo rápido):

Restaurar dependências: dotnet restore
Entrar na pasta do projeto: cd API
Executar a API: dotnet run
(Opcional) Aplicar migrações: dotnet ef database update (requer EF tools)
Observações finais:

Código e configuração foram desenvolvidos para fins educacionais; use com cuidado em ambientes reais.
